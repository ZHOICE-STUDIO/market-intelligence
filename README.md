# market-intelligence

Game market research and data analysis repository for ZHOICE STUDIO.

## Structure

```
market-intelligence/
├── data/
│   ├── raw/          # Source data (large files excluded from git)
│   └── processed/    # Cleaned and transformed data
├── notebooks/        # Jupyter notebooks for analysis
├── reports/          # Findings, charts, and summaries
└── src/              # Reusable scripts and helpers
```

## Topics

- Global game market overview
- Genre and platform trends
- Competitor analysis
- Player demographics and behavior
- Revenue model benchmarks
- Distribution channel comparison
